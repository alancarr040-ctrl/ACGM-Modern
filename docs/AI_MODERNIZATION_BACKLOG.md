# ACGM Modern Client Modernization Backlog

This backlog contains ideas intentionally deferred until Phase 0.12 or later. These items are not part of 0.11 legacy certification unless the VB6 client is later proven to have supported the behavior.

## Workspace Improvements

- Double-click any character list opens the Character Dialog.
- Apply consistent list navigation behavior to Flat Listing, Rescue Squad, Summonable Portal List, Trade Skills List, Search Results, and future character-list workspaces.

## Search

- Live filtering.
- Export results.
- Saved searches.
- Column sorting.
- Search history.
- Additional search criteria.

## Character Dialog

- Auto-load Trade Skills when the tab/list is first opened.
- Auto-load Rescue Squad when the tab/list is first opened.
- Auto-load Portal List when the tab/list is first opened.

## Administration

- Delete Character, administrator only.
- Rename Character, administrator only.
- Merge Character, administrator only.
- Character repair and validation tools.

## Performance

- Cached search index.
- Cached parsed skill index for faster skill filtering.

## Diagnostics

- Optional broader protocol tracing.
- Optional tree refresh diagnostics.
- Optional search performance timing.
